# Supermarket Sales Dashboard

## Project Overview
**This project is a Power BI dashboard created using supermarket sales data.
The dashboard helps analyze:

1.Total Sales
2.Profit
3.Product Performance
4.Customer Trends
5.Payment Methods
6.City-wise Sales

Tools Used
- Power BI
- Excel / CSV
- DAX
- MySQL


Dashboard Features
- KPI Cards
- Sales Trend Analysis
- Category-wise Performance
- Filters & Slicers
- Interactive Visualizations

Project Workflow
1. Imported supermarket sales dataset into MySQL
2. Connected MySQL database with Power BI
3. Performed data cleaning using Power Query
4. Created KPIs and DAX measures
5. Built interactive dashboard visualizations
6. Generated business insights from sales data


Datasets
- Branch
- City
- Customer_type
- Gender
- Invoice_ID
- gross_income
- Quantity

Key Insights
- Total Sales reached 15.38K.
- Food and Beverages generated highest gross income.
- Customer ratings were consistently above 6.8 across all product categories.
